# Рабочий резерв — лендинг

Одностраничный сайт услуг разнорабочих в Челябинске: HTML, CSS и JavaScript без сборки.

## Возможности сайта

- Приветственный блок с призывом к действию
- Навигация по разделам и подсветка текущего раздела при прокрутке
- Калькулятор ориентировочной стоимости (часы × ставка)
- Переключение светлой и тёмной темы (сохраняется в браузере)
- Форма заявки с проверкой данных и открытием Telegram для отправки текста
- Адаптивная вёрстка и учёт `prefers-reduced-motion`

## Структура репозитория

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` | Разметка страницы |
| `styles.css` | Стили, темы, адаптив |
| `script.js` | Форма, тема, калькулятор, меню, навигация |

## Быстрый запуск

Откройте `index.html` в браузере (двойной клик или перетаскивание в окно браузера).

## Редактирование

- Тексты и блоки — в `index.html`
- Цвета темы (`:root` и `[data-theme="light"]`) — в `styles.css`
- Логика формы, темы и калькулятора — в `script.js`

## Заявка через форму

После нажатия «Отправить» проверяются имя и телефон, формируется текст заявки и открывается окно Telegram с готовым сообщением; пользователь отправляет его вручную.

## Политика и согласия (Supabase)

Для "легально-достаточного" хранения согласий и версии политики выполните SQL-скрипт:

- `supabase_privacy_setup.sql` — добавляет поля согласия в `orders`, таблицы `policy_versions` и `admin_settings`, а также RLS-политики для внутренних настроек админки.

Рекомендуется хранить в каждой заявке:

- `privacy_consent` (факт согласия),
- `consent_at` (время согласия),
- `policy_version` (версия политики на момент отправки),
- `policy_url` (ссылка на редакцию).
